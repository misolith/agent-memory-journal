# Quickstart

## Add a note

```bash
memory-recall-guard add --note "Remember to renew PAT before Friday"
```

## Add a note to long-term memory too

```bash
memory-recall-guard add --note "Use playing golf app path for live tee time checks" --long
```

## Show recent notes

```bash
memory-recall-guard recent --days 2
```

## Search notes

```bash
memory-recall-guard search --query "golf"
```

## Review recurring patterns

```bash
memory-recall-guard topics --days 14
memory-recall-guard cadence --days 14
memory-recall-guard digest --days 7
memory-recall-guard candidates --days 7
```
